YT Cutter

Extract favorite scenes from YouTube videos

You can't download your clip from this website, but here are some instructions how to create it on your own:

  1. Run cmd.exe - it'll open a console window. Enter: mkdir C:\YTcutter This will create a YTcutter folder on C: drive where you'll keep all the required files.
  2. Then type: cd C:\YTcutter To go into that folder.
  3. Download YT-DLP which you'll use to download videos from youtube.
    Move the downloaded file to C:\YTcutter. Now go back to your console window and enter: yt-dlp You should see some usage info from yt-dlp. If you see something like "command not found" check if your yt-dlp.exe file is in C:\YTcutter folder.
  4. Similarily download FFMPEG which you'll use to cut fragments from downloaded videos (direct link for Windows OS).
    In the downloaded archive you're only interested in the bin\ffmpeg.exe file. Extract that file to C:\YTcutter.
    Go back to your console window and type: ffmpeg You should see some text output from ffmpeg. If you see something like "command not found" check if your ffmpeg.exe file is in C:\YTcutter folder.
  5. If everything works correctly, go to your console window and enter (copy & paste it): yt-dlp "https://www.youtube.com/watch?v=ksjHzA1QAcg" -o ksjHzA1QAcg.webm This will download the entire video to your computer. If there's some error, make sure you have the latest version of yt-dlp (it changes pretty often)
  6. Now run ffmpeg to cut your fragment: ffmpeg -ss 100 -t 60 -i ksjHzA1QAcg.webm -y -c copy video-100-60-ksjHzA1QAcg.mp4
    Your fragment is now in folder C:\YTcutter in file video-100-60-ksjHzA1QAcg.mp4
    Congratulations.

    If you want more clips from this video, next time skip steps 1,3,4,5 and only run commands from step 2 and 6.
    If you're done, you can delete the downloaded video to save disk space: del ksjHzA1QAcg.webm



Didn't work for you? Here are some other methods:
  • Record your screen while playing your YT video.
  • Record whatever audio is playing on your computer.
  • Just press Print Screen (Prt Scr) key while playing your video to make a screenshot of your screen.


In any case ask YT and video author for a permission before you save a video on your disk.
